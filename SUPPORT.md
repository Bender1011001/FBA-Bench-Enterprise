# Support

## Documentation

- Project docs: `docs/README.md`
- Developer setup: `DEV_SETUP.md`

## Issues

Use GitHub Issues for:
- bug reports
- feature requests
- documentation fixes

Email support:
- `support@fbabench.com` (general support and commercial licensing inquiries)

Include:
- reproduction steps
- logs (redact secrets)
- environment details (OS, Python version, Poetry version)

## Security

For vulnerabilities, follow `SECURITY.md` instead of filing a public issue.
