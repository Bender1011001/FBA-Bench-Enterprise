## Issues

Use GitHub Issues for:
- bug reports
- feature requests
- documentation fixes

Include:
- reproduction steps
- logs (redact secrets)
- environment details (OS, Python version, Poetry version)

## Security

For vulnerabilities, follow `SECURITY.md` instead of filing a public issue.
