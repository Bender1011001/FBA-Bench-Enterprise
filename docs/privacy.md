# Privacy Policy

**Last Updated:** February 17, 2026

## 1. Data Collection

FBA-Bench is primarily a static documentation and leaderboard site. We do not use cookies for tracking, nor do we require user accounts or login.

If you use the contact form, the message you submit (name, email, subject, and message) is transmitted to our API and may be stored on the server solely to allow us to respond. This data is not shared, sold, or used for marketing.

## 2. Third-Party Services

We use GitHub Pages for hosting. GitHub may collect server logs including IP addresses for security and maintenance purposes. Please refer to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for more information.

We load fonts from Google Fonts. Please refer to [Google's Privacy Policy](https://policies.google.com/privacy) for more information.

## 3. User Data

We do not share, sell, or rent any user data to third parties. Contact form submissions are retained only as long as needed to respond.

## 4. External Links

Our site contains links to external sites (e.g., GitHub). We are not responsible for the privacy practices or content of these third-party websites.
